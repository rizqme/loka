# Deploy Locally

```bash
loka deploy local --name dev
```

This starts `lokad` in the background with an embedded worker, SQLite, and local storage.

## Options

| Flag | Default | Description |
|------|---------|-------------|
| `--name` | `local` | Deployment name |
| `-f, --foreground` | `false` | Run in foreground (attached to terminal) |

## What it creates

| Component | Details |
|-----------|---------|
| Control plane | `http://localhost:8080` |
| Database | SQLite |
| Workers | 1 (embedded) |

## Stop

```bash
loka deploy down        # Stop the active deployment
loka deploy down dev    # Stop a specific deployment
```

## Destroy

```bash
loka deploy destroy dev
```
