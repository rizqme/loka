# Self-Managed Workers

Connect any Linux machine with KVM to a LOKA deployment.

## Requirements

- Linux with `/dev/kvm`
- Network access to control plane (port 9090)

## Setup

### 1. Start a control plane

```bash
loka deploy local --name my-cluster -f
```

### 2. Create a registration token

```bash
loka token create --name my-server
# Token: loka_a2d9...
```

### 3. Install and start the worker

```bash
ssh user@your-server
curl -fsSL https://rizqme.github.io/loka/install.sh | bash

cat > /etc/loka/worker.yaml << YAML
control_plane:
  address: "<cp-host>:9090"
token: "loka_a2d9..."
labels:
  gpu: "true"
YAML

loka-worker
```

### 4. Verify

```bash
loka worker list
```
